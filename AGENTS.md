# A2 Media Storage Instructions

- This repo is static media storage for the Alex birthday site.
- Never commit the source ZIPs or raw temporary extraction folders.
- Strip EXIF and GPS metadata from generated public images.
- Keep JSON paths relative to the A2 repository root.
- Do not change the public folder structure casually because Alex depends on it.
- Validate that every manifest path exists before publishing.
- Keep the output GitHub Pages friendly and dependency-light.
